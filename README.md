
# Week 1 - Network Administration Internship

## Networking Fundamentals & Network Simulation Tools

### Objectives

* Learn networking fundamentals
* Understand IP addressing
* Explore Cisco Packet Tracer and GNS3
* Build a simple LAN topology
* Verify connectivity using ping

### Tools Used

* Cisco Packet Tracer
* GNS3
* PuTTY

### Topology

A simple LAN was created and configured using static IP addressing.

### Verification

Connectivity between devices was tested successfully using the ping command.
